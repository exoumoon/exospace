### Usage

###### Prerequisites

```nushell
# 1. Clone the repo:
git clone https://github.com/exoumoon/exospace
cd exospace

# 2. Make sure you have these tools:
docker         # https://www.docker.com
docker compose # https://docs.docker.com/compose
invar          # https://crates.io/crates/invar
```
